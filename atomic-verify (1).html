Protocol Architecture

A deterministic 7-step flow that eliminates human intervention from the dispute lifecycle.

1

User initiates payment

React.js frontend calls depositFunds() on the smart contract

2

Funds enter escrow

Smart contract locks funds in "Committed" state, starts 60s timer

3

Oracle pings merchant API

Chainlink Oracle sends GET request to Node.js merchant backend

4

Decision: 200 OK or failure?

Contract executes releaseFunds or refundUser

// Execution Flowchart
🖥️ User clicks "Subscribe" on React Dashboard
📝 Frontend calls depositFunds() on Smart Contract
🔒 Funds locked → State: COMMITTED | Timer: 60s
🔗 Chainlink Oracle → POST request to Node.js Merchant API
⏱️ Does Merchant API return 200 OK within 60s?
✓ YES — Success
✅ Oracle sends "Success Hash" → releaseFunds() → Merchant wallet credited
✗ NO / TIMEOUT / 500
❌ Oracle sends "Failure" → Timer expires → refundUser() → Instant revert
